Para tener en cuenta

Una de las cosas que debemos tener claras es que al momento de usar esta api es que todas las peticiones que hagamos deben apuntar al dominio:https://api-the-movie-nodejs.herokuapp.com/api/peliculas una vez claro esta parte podemos empeza.

Peliculas recientes

Si desea obtener las peliculas las recientes, se debe hacer una peticion con el metodo GET a la direcion /recientes de esta forma podra obtener las peliculas mas recientes.

https://api-the-movie-nodejs.herokuapp.com/api/peliculas/recientes

Peliculas con la mejor calificaion

Para obtener las peliculas con las mejores calificaion se debe hacer una petecion con el metodo GET a la direccion /calificacion las peliculas que son obtenidas de esta forma se encuentran organizadas de manera descendente.

https://api-the-movie-nodejs.herokuapp.com/api/peliculas/calificacion

Peliculas populares

Para obtener las peliculas mas populares se debe hacer una petecion con el metodo GET a la direccion /populares las peliculas que son obtenidas de esta forma se encuentran organizadas de manera descendente.

https://api-the-movie-nodejs.herokuapp.com/api/peliculas/populares

Buscar peliculas

Para realizar la busqueda de una pelicula debe hacerce por le metodo POST a la direccion /buscar se debe enviar un json con la siguiente estructura:

{ "buscar": "Palabra o titutlo de la policula" }

https://api-the-movie-nodejs.herokuapp.com/api/peliculas/buscar

Obtener detalle de una pelicula

Para realizar la busqueda de una pelicula debe hacerce por le metodo GET a la direccion /pelicula tambien debe enviar en el id de la pelicula como parametro en la url

https://api-the-movie-nodejs.herokuapp.com/api/peliculas/pelicula/{id de la pelicula}
